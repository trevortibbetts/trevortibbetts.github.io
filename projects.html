Research Projects

Spatial Electoral Polarization

MS Thesis project examining precinct-level election outcomes using spatial statistics. This work applies the S-statistic to measure partisan clustering across multiple geographic scales.

Methods: Python, PySAL, precinct-level election data
Status: MS Thesis / Conference Presentation

GitHub repository

Drag Queen Migration Map

A spatial analysis of drag performer migration patterns, exploring how cultural labor, urban scenes, and regional mobility intersect within queer performance networks.

Methods: Python, GIS, spatial visualization
Theme: Cultural geography & mobility

GitHub repository

Gerry SD

An interactive project that allows users to experiment with gerrymandering in San Diego County, designed to make redistricting concepts accessible to a public audience.

Focus: Civic education, spatial interaction, UI design
